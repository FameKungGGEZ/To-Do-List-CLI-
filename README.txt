📌 To-DO List CLI (Command Line Interface)

📑 คำอธิบาย:
โปรแกรมใช้สำหรับบันทึกสิ่งต่างๆที่ต้องทำ (To-Do List) ผ่านทาง Command Line โดยบันทึกข้อมูลในรูปแบบ ไฟล์ tasks.json

🛠 วิธีใช้งาน :
1.เมื่อเปิดโปรแกรมจะมีเมนูให้เลือกทั้งหมด4อย่าง
    -1.เพิ่มงานใหม่
    -2.ลบงาน
    -3.แสดงงานทั้งหมด
    -4.ออกจากโปรแกรม

2.โปรแกรมนี้จะเก็บไฟล์ทั้งหมดหมดที่คุณได้เพิ่มงานไปไว้ใน ไฟล์ tasks.json โดยอัตโนมัต

💡 หมายเหตุ :
- ถ้าไฟล์ tasks.json เสียหรือมีปัญหาใช้งานไม่ได้ โปรแกรมนี้จะสร้างใหม่ทันที โดยอัตโนมัต
- ถ้าปิดไฟล์กระทันหันจะไม่มีการเซฟ
- title ของlistสามารถซ้ำกันได้ดังนั้นโปรดจำให้ดีว่าอันไหนเป็นอันไหนถ้าตั้งซ้ำ
